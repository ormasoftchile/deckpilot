# Welcome to Deckpilot Sidecar

Deckpilot supports a **dual authoring model**: keep your Markdown clean and readable,
while storing all operational metadata — actions, cues, timing, and recording settings —
in a companion `.deck.yaml` sidecar file.

---

<!-- id: setup -->

# Setup

Install the project dependencies before running anything:

```bash
npm install
```

---

<!-- id: demo -->

# Run the Demo

Once dependencies are installed, start the application:

```bash
npm start
```
