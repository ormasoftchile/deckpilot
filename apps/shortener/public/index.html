d3ck.md

Short URLs for Deckpilot decks.

GitHub

https://d3ck.md/gh/<owner>/<repo>/<path>
https://d3ck.md/gh/<owner>/<repo>@<branch|tag|sha>/<path>

Resolves raw.githubusercontent.com/<owner>/<repo>/<ref>/<path> and opens it in the viewer.

Gist

https://d3ck.md/gist/<user>/<id>
https://d3ck.md/gist/<user>/<id>/<file>

GitLab

https://d3ck.md/gl/<owner>/<repo>/<path>
https://d3ck.md/gl/<owner>/<repo>@<ref>/<path>

Arbitrary URL

https://d3ck.md/u/<base64url-of-https-url>

Escape hatch for any HTTPS-hosted .deck.md.
